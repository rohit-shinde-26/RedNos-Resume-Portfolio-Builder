# 🚀 RedNos Resume & Portfolio Builder

An AI-powered web application to create professional **resumes** and **developer portfolios** and instantly. This project is built using **HTML, CSS, and JavaScript**, focusing on simplicity, speed, and user experience.

## 🌐 Live Demo

👉 [https://red-nos-resume-portfolio-builder.vercel.app/](https://red-nos-resume-portfolio-builder.vercel.app/)

---

## ✨ Features

* 🤖 AI-powered resume & portfolio content generation
* 📄 Instant resume creation
* 🌐 Personal portfolio generator
* 🎨 Clean and responsive UI
* ⚡ Fast performance (no heavy backend)
* 🧑‍💻 Beginner-friendly and lightweight

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **AI Integration:** API-based (if used)
* **Deployment:** Vercel

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/rohit-shinde-26/rednos-resume-builder.git
```

Navigate into the project:

```bash
cd rednos-resume-builder
```

Open in browser:

```bash
index.html
```

---

## ⚙️ Configuration

If you are using an AI API, create a `.env` file (or directly configure in JS for local testing):

```env
API_KEY=your_api_key
```

> ⚠️ Note: Do not expose API keys in public repositories.
